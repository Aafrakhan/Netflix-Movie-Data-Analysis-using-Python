# Netflix Movie Data Analysis using Python

## Overview
An end-to-end Exploratory Data Analysis (EDA) project on Netflix movie data using Python. The dataset was cleaned and processed to ensure accuracy and consistency, followed by detailed analysis to uncover trends in genre popularity, audience ratings, and release patterns over the years.

## Tools & Libraries Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Columns include: Release_Date, Title, Overview, Popularity, Vote_Count, Vote_Average, Original_Language, Genre, and Poster_Url.

## Analysis Performed
- Data cleaning and preprocessing of raw movie dataset (`mymoviedb.csv`)
- Genre frequency analysis across the dataset
- Popularity and vote-based analysis to identify top and bottom performing movies
- Trend analysis of movie releases across decades (1900–2020)
- Visualization of genre distribution and rating patterns using bar charts and histograms

## Key Insights (Q&A Format)
- **Most frequent genre:** Drama, appearing in more than 14% of entries among 19 other genres
- **Highest-voted genre:** Drama again leads, featured in over 18.5% of the 25.5% of movies with popular votes (6,520 rows)
- **Highest popularity movie:** *Spider-Man: No Way Home* (Genres: Action, Adventure, Science Fiction)
- **Lowest popularity movie:** *The United States, Thread* (Genres: Music, Drama, War, Sci-Fi, Historic)
- **Most active filming year:** 1970 recorded the highest number of filmed movies in the dataset
